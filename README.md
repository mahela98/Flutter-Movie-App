## Movie App using Flutter and MovieDB API

### Functions
- Trending Movies
- Top Rated movies
- Upcomming Movies
- Popular TV shows
- Top TV shows
- Search Movies by name

### Technologies Used
- Flutter
- MovieDB API
- Youtube API
- VS-Code
- Postman
